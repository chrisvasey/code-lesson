Week 1 - Part 2: Diving into variables

What is a variable?

Veriables are bits of data that we assign names to, this way we can do things with them like output them, change them or save them to a database. We are basically helping our code remember things.

In JavaScript these can be declared using the word var (short for variable?)
You almost always you want to start variable names with lowercase letters or things get a bit weird.
You use a = sign to tell it what data to assign e.g
var greeting = "Hello";

Data types

Variables can be assigned a few different types of data, don't worry too much about this, we will go into them as we need them but just so you have an overview these are the possible data types:

  1. String - Normally text but can also include numbers
  2. Number (Integer/Float) - Basically just numbers
  3. Boolean - This either true or false
  4. Object - A way to structure data which is made of of "Key/Pairs". Objects are wrapped in {} brackets
  5. Array - A collection of the above, can also be a collection of arrays. Arrays are wrapped in [] brackets
  6. Function - A is a block of code designed to perform a particular task.
