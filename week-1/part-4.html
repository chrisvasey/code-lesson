Week 1 - Part 4: Conditionals and Comparison

Conditionals (If Statements)

Most people may have come across conditonals before, they are methods for deciding when code should run or not.
The most basic and widely used conditonal is an "If" statement, you may have used these before in programs like Excel.
If statements check to see if something is equal to "true" and then runs the code. In JavaScript they look like this:
var friend = true;
if(friend){
    console.log("Not foe!");
}
You can also handle what to do when an IF statement isn't going to run using "else"
var admin = false;
if(admin){
    //Give them access
    accessMainframe();
} else {
    //Banned!
    banHammer();
}
Here we are checking to see if the "admin" veriable is true

Operators

In JavaScript we have some symbols that help us work with data, they help us compare and check values.
Some examples of these are:

  • == - Check if two items are the same or "equal"
  • != - Check if two items are diffrent or "not equal"
  • > - Checks if a value is greater than
  • < - Check if two items are less than
  • >= - Check if two items are less than
  • <= - Check if two items are less than
