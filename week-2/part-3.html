A quick recap on IF/Else statements

Very often when you write code, you want to perform different actions for different decisions.
You can use conditional statements in your code to do this.
Here is an example of an IF/Else statment 

						
							if (condition) { 

							  //  block of code to be executed if the condition is true 

							} else { 

							  //  block of code to be executed if the condition is false 

							}

Operators

In JavaScript we have some symbols that help us work with data, they help us compare and check values.
Some examples of these are:

  • == - Check if two items are the same or "equal"
  • != - Check if two items are diffrent or "not equal"
  • > - Checks if a value is greater than
  • < - Check if two items are less than
  • >= - Check if two items are less than
  • <= - Check if two items are less than
