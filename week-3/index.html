  • Loops can execute a block of code a number of times, they are handy if you need to run the same code over and over again e.g for counting.
    There are a few different types of loops which we will go through later but the most common one is the for loop.
    The syntax looks like this (it can be a bit confusing at first): 
    							
								for (statement 1; statement 2; statement 3) {
								  // code block to be executed
								}
    Statement 1 is executed (one time) before the execution of the code block.
    Statement 2 defines the condition for executing the code block.
    Statement 3 is executed (every time) after the code block has been executed.
    Some examples of real for loops: 
    								
									//Counting to 100
									for (i = 1; i < 101; i++) {
									  console.log("The number is " + i);
									}
									
									//Looping though an Array
									var cats = ['Jimmy', 'Jane', 'Sally'];
									for (var i = 0; i < cats.length; i++) {
										console.log("The cats name is: "+cats[i])
									}
